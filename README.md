# ✨ Hand-Tracked Particle Visualization

An interactive 3D particle visualization built using **Three.js** and **MediaPipe Hands**. Control thousands of particles in real-time using hand gestures through your webcam.

## 🚀 Features

- 🖐️ Real-time hand tracking using MediaPipe
- 🌌 6000+ animated particles
- 🔵 Multiple particle shapes:
  - Sphere
  - Spiral
  - Torus
  - Heart
- 🤏 Pinch gesture for zoom control
- ✋ Hand movement repels particles
- 👊 Fist gesture switches particle shapes
- 🎨 Dynamic cyberpunk particle colors
- ⚡ Smooth particle physics and animations
- 📷 Live webcam integration

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Three.js
- MediaPipe Hands
- Camera Utils

## 📸 Demo Controls

| Gesture | Action |
|----------|---------|
| Pinch (Thumb + Index) | Zoom In / Out |
| Move Hand | Repel Particles |
| Make Fist | Switch Shape |

## ▶️ How to Run

1. Clone the repository

```bash
https://github.com/priya16150/3-JS-PARTICLE----3d-effects/tree/main
```

2. Open the project folder

```bash
cd hand-tracked-particles
```

3. Run with Live Server

- Open in VS Code
- Install Live Server Extension
- Click "Go Live"

OR

Use any local web server.
